### flying-buddha

little browser game

saves the highscore list in highscore.txt on the server

#### dependencies
written in python 2.7. needs (pip install):
+ flask
+ flask-httpauth
+ jinja2
+ werkzeug
+ argparse
+ distribute
+ flup
+ wsgiref
